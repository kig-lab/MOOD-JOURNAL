**MOOD-JOURNAL**
A simple Python-based mood journaling app that helps users record their daily moods and notes, then view summaries and trends over time.

#FEATURES

Add daily mood entries with optional notes

View a weekly mood summary

Visualize mood trends using charts

Simple, beginner-friendly Python logic

##TECHNOLOGIES USED

Python

Matplotlib (for mood charts)

Standard Python libraries (datetime, collections)

##HOW TO RUN

Clone the repository

Make sure Python is installed

Run the main Python file: terminal: python app.py

##PURPOSE

This project was built to practice Python programming, data handling, and basic data visualization while promoting mental health awareness.
